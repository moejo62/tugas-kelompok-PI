Event Keterangan
show Event ini dipicu segera setelah instan dari method show dipanggil
shown Event ini dipicu ketika sebuah collapsible elemen dibuat visible ke user (menunggu CSS transition selesai).
hide Event ini dipicu segera ketika hide method telah dipanggil.
hiden Event ini dipicu ketika sebuah collapsible elemen telah disembunyikan dari user (akan menunggu sampai CSS transition selesai).
