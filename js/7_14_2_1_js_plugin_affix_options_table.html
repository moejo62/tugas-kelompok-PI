Options Type Default Keterangan
offset number | function | object 10 Jarak kosong (offset) dalam pixel ketika menghitung posisi scroll. Bila angka tunggal digunakan, maka offset akan diterapkan baik dari arah atas maupun dari arah kiri. Untuk menerapkan satu arah tunggal, atau multiple unique offset, gunakan object offset: { x: 10 }. Gunakan sebuah fungsi jika kita ingin memberikan nilai offset secara dinamis.
